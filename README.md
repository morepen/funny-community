## 开发环境
 [Node.js](https://nodejs.org) 版本  v6.11.0，[MongoDB](https://www.mongodb.org) 版本是 v3.4.5
```
1. 环境中必须有 Node.js 和 MongoDb
2. 启动 MongoDB 数据库
3. npm install 安装项目依赖包
4. 根据需求修改配置文件 config.default.js => config.js
5. 运行 app.js
6. 访问 127.0.0.1:3000
```
